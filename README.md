## Build and Install

```
$ go install github.com/spiegel-im-spiegel/linkcard@latest
```

## Binaries

See [latest release](https://github.com/spiegel-im-spiegel/linkcard/releases/latest).

## Usage

```
$ linkcard [flags] <url> [<url> ...]
```
